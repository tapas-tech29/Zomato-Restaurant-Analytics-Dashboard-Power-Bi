## 📌 Project Overview

The **Zomato Restaurant Analytics Dashboard** is an interactive Power BI project designed to analyze restaurant performance across multiple Indian cities. The dashboard provides actionable insights into customer ratings, cuisine preferences, restaurant types, pricing trends, online ordering behavior, and customer engagement.

This project demonstrates practical skills in **Data Cleaning, Data Modeling, DAX Calculations, Data Visualization, and Business Intelligence Reporting** using Power BI.

---

## 🎯 Project Objectives

* Analyze restaurant performance across different cities
* Compare cuisine popularity and customer ratings
* Evaluate online ordering and customer engagement trends
* Understand cost versus rating relationships
* Create an interactive dashboard for business decision-making

## 💡 Key Insights

* Online ordering is adopted by nearly 75% of restaurants.
* Ahmedabad has the highest number of restaurants in the dataset.
* Jaipur records the highest average restaurant ratings.
* Continental and South Indian cuisines receive the strongest customer ratings.
* Budget restaurants perform competitively against premium restaurants.
* Mid-range restaurants provide the best balance between affordability and popularity.
* Customer engagement is influenced by both ratings and vote volume.
